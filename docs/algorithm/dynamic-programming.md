## 状态转移方程

状态转移方程是动态规划中的一个核心概念，它描述了问题求解过程中不同阶段状态之间的关系。

<div class="formula">
F(n) = F(n−1) + F(n−2)，n ≥ 3，F(1) = 1，F(2) = 1
</div>
